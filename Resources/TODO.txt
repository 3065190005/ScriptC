# 语法修改
1. new <interface> +> new <interface>(<args...>)	通过指定参数进行构造(不可主动调用)
2. interface => struct					将接口改为结构体
3.1 +co_resume(<coroutineId>, <value>) 			通过id恢复一个协程并将参数value发送到协程
3.2 +co_yeild(<value>)					暂停一个协程并返回value
3.3  +co_cancel(<coroutineId>)				关闭一个协程



# 接口特殊方法
1. _init(<args...>) 			初始化调用
2. _attr(<name>, <type>, <list>)	当调用不存在的方法或属性时调用
3. _call(<args...>)			当作为函数调用时自动进入
