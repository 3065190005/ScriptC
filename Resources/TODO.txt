# 语法修改
1. new <interface> +> new <interface>(<args...>)	通过指定参数进行构造(不可主动调用) (Complate)
2. interface => struct					将接口改为结构体
3.1 +resume(<coroutineId>, <value>) 			通过id恢复一个协程并将参数value发送到协程 (Complate)
3.2 +yeild(<value>)					暂停一个协程并返回value (Complate)



# 接口特殊方法
1. _init(<args...>) 			初始化调用 (Complate)
2. _attr(<args...>)			当调用不存在的方法或属性时调用	-> 变量 (Complate) && 函数 (Complate)
3. _call(<args...>)			当作为函数调用时自动进入
